# CEIN Camping

A mobile-friendly e-commerce style website for outdoor products, built with Next.js and powered by PostgreSQL.  
Add and filter product listings, categorize items, manage via a database or through the site.

**Live demo:** [https://cein-camping.vercel.app](https://cein-camping.vercel.app)  
**Repository:** [alexorno/CEIN-camping](https://github.com/alexorno/CEIN-camping)

## Features

- Multi-category outdoor products (camping gear, etc)  
- Filtering functionality (by category or other attributes)  
- Admin / data management via database or site interface  
- Fully responsive, mobile-friendly layout  
- Built with performance and maintainability in mind  

---

## Tech Stack

- Front-end & SSR: **Next.js**  
- Database: **PostgreSQL**  
- UI: React, CSS (or styled components)  
- Deployment: Vercel  
- Languages: JavaScript, HTML5, CSS3

## Getting Started

### Prerequisites

- Node.js (v16+ recommended)  
- npm or yarn  
- PostgreSQL instance (local or remote)  

### Installation

```bash
# Clone the repo
git clone https://github.com/alexorno/CEIN-camping.git
cd CEIN-camping

# Install dependencies
npm install
# or
yarn install
